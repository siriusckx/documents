## 让程序不能被多次进程
### 问题
>守护进程是长时间运行而不退出，每执行一次，就有多一个进程。这不是我们需要的，还有可能带来错误。
### 实现方法
1. 最常用的一种方法：用一个文件的存在与否来做标志。具体做法是程序在执行之初检查一个文件是否存在，存在则不启动，不存在则启动创建一个文件。程序执行完以后，需要删除文件。
```
fd = open(FILE,O_RDWR|O_TRUNC|O_CREATE|O_EXCEL, 0664);
if(fd < 0)
{
    if(errno == EXIST)
    {
        printf("文件已经存在!");
        exit(-1);
    }
}

remove(FILE);
```
