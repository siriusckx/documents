## 变量的赋值与引用
### 定义新的变量
1. 变量以字母或_开头，区分大小写
2. 格式：变量名 = 变量值
### 查看变量的值
1. echo $变量
```
echo $name
echo ${name}fg
```
2. 从键盘为某个变量赋值
```
read -p "please input your name:" name
echo $name
```
3. 结合不同的引号在脚本中的使用
   * 解析变量时要用双"号
   * 不需要解析变量单'号
   * 解析命令用返`号：执行命令的结果返回给变量
