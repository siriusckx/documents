### 作用
> 远程连接到指定的服务器。

1. 使用22端口，当前用户名远程连接
    ```
    ssh 192.168.1.10
    ```
1. 指定用户名，指定端口进行远程连接
    ```
    ssh -l root 192.168.1.10 -p 4000
    或者
    ssh root@192.168.1.10 -p 4000
    ```
1. 连接的时候，显示调试信息
    ```
    ssh 192.168.1.10 -v
    ```
1. 连接上以后，执行某个命令
    ```
    ssh 192.168.1.10 ls
    ```
