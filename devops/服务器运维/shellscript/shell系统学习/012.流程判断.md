## 流程判断
### 单分支
```
age = 18
if [ $age -lt 18]
then
    echo ""
else
    echo ""
fi
```

### 多分支
1. if语句
```
score = 61
if [ $score -lt 60 ];then
elif [ ];then
    echo ""
elif [ ];then
    echo ""
else
    echo ""
fi
```

2. case语句
```
week = `date +%w`
case $week in
    1)
    echo ""
    ;;
    2)
    echo ""
    ;;
    *)
    echo "other"
    ;;
esac
```

```
case $1 in
    start)
    echo "start"
    ;;
    stop)
    echo "stop"
    ;;
    restart)
    echo "restart"
    ;;
    *)
    echo "start|stop|restart"
    ;;
esac
```
