# 一台服务器到其他服务器设置免密登录
```
vim /etc/hosts
127.0.0.1   localhost
10.10.0.100 s100
10.10.0.101 s101

ssh-keygen -t rsa
ssh-copy-id -i /root/.ssh/id_rsa.pub -p 43322 root@s100
ssh-copy-id -i /root/.ssh/id_rsa.pub -p 43322 root@s101
```
